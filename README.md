# discord-webhook
DiscordのWebhookを活用した便利そうなアプリのソースコード

--- 

## Webhookとは...
DiscordのWebhookは、「外部サービスからの通知をDiscordへ自動転送するための専用窓口」です。  
本来、Discordで発言するには人間が操作するか、複雑なプログラム（Bot）を組む必要がありますが、Webhookを使えば発行された専用のURLにデータを送るだけで、誰でも簡単に特定のチャンネルへメッセージを表示させることができます。  

--- 

## DiscordのWebhook URLを取得する手順
1. チャンネル設定を開く: 投稿したいチャンネル名の横にある歯車アイコン（チャンネルのエディット）をクリックします。
2. 連携サービスを選択: 左メニューから「連携サービス」を選択します。
3. Webhookを作成: 「ウェブフックを作成」ボタン（または「ウェブフック」項目）をクリックします。
4. 設定を保存: 必要に応じて名前やアイコンを変更し、「変更を保存」をクリックします。
5. URLをコピー: 画面内にある「ウェブフックURLをコピー」ボタンをクリックして完了です。

※ 注意: このURLを知っている人は誰でもそのチャンネルに投稿できてしまうため、他人に教えないよう厳重に管理してください。

## Pythonアプリを裏で常時稼働させる手順
### 1. アプリをバックグラウンド化する
黒い画面（コンソール）を出さずに、裏側だけで静かに動くようにします。

1.  作成した Pythonファイル（`.py`）の拡張子を名前変更で **`.pyw`** に変えます。
    *   例: `app.py` → `app.pyw`
2.  そのファイルを**ダブルクリック**します。
    *   画面には何も出ませんが、これで起動完了です。
    *   止めたい時は、タスクマネージャーから `Python` 関連のプロセスを終了させます。

### 2. PC起動時に自動で動くようにする（スタートアップ）
PCの電源を入れるたびに手動で起動しなくて済むようにします。

1.  先ほど作った `.pyw` ファイルを右クリックし、**「ショートカットの作成」** を選びます。
2.  キーボードの **[Windows] + [R]** を押し、出てきた画面に `shell:startup` と入力して OK を押します。
3.  開いたフォルダの中に、さっき作った**ショートカット**を入れます。
    *   これで次回のPC起動時から勝手に動き出します。

### 【オプション】管理者権限で動かしたい場合
キーボード操作を監視するアプリなど、**「強い権限」**が必要な場合だけこの手順を行います。
（※これを行うと、PC起動時に毎回「許可しますか？」という確認画面が出るようになります）

1.  作成したショートカットを右クリックして **「プロパティ」** を開きます。
2.  **「ショートカット」タブ** にある **「リンク先(T)」** を書き換えます。
    *   元々のパスの**先頭に** `pythonw.exe ` （最後に半角スペース）を追記します。
    *   例: `"C:\...\app.pyw"`
    *   ↓↓↓
    *   変更後: `pythonw.exe "C:\...\app.pyw"`
    *   **「適用」** を押します。
3.  同じ画面にある **「詳細設定(D)...」** ボタンを押します。
4.  **「管理者として実行(R)」** にチェックを入れて OK を押します。
5.  最後にプロパティ画面も OK で閉じれば完成です。
